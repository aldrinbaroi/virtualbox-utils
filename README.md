# virtualbox-utils
Utility scripts for VirtualBox

```
Usage: 
        vm-manager (-c|-r) VM_CONFIG_FILE | -d VM_NAME | -h"

        -c Create VM"
        -r Recreate VM.  Existing VM will be delete."
                         Running VM will be forcefully shutdown & deleted"
        -d Delete VM"
        -h Show this usage text"
```
